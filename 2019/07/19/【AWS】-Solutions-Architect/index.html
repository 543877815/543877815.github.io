持续集成

持续集成 是一种开发运营软件开发实践，通过该实践，开发人员可定期将代码更改并入中央存储库，然后运行自动化构建和测试。持续集成通常是指软件发布过程的构建或集成阶段，同时包含自动化组件（例如，CI 或构建服务）和文化组件（例如，学习频繁集成）。持续集成的主要目标是更快地查找和解决错误、提高软件质量以及减少验证和发布新软件更新所需的时间。

持续交付

持续交付 是一种开发运营软件开发实践，通过该实践可以自动构建、测试和准备代码更改以便投产。它通过在构建阶段之后将所有代码更改部署到测试环境和/或生产环境来扩展持续集成。在正确实施持续交付时，开发人员将始终拥有已通过标准化测试流程的部署就绪构建构件。

持续交付使开发人员能够实现测试自动化，而不仅仅是单元测试，这样他们便可以在部署到客户之前跨多个维度验证应用程序更新。这些测试可能包括 UI 测试、负载测试、集成测试、API 可靠性测试等。这有助于开发人员更彻底地验证更新并提前发现问题。借助云，可以轻松且经济高效地自动创建和复制用于测试的多个环境，而这以前很难在本地完成。

您知道蓝/绿开发意味着什么吗？ 构建和部署软件的其他策略是什么？

蓝/绿部署是发布软件代码的变更管理策略。 蓝/绿部署（也可以称为 A/B 部署）需要两个相同的硬件环境（即配置方式完全相同）。当一个环境处于活动状态并为最终用户提供服务时，另一个环境则保持空闲状态。

蓝/绿部署通常用于面向消费者的应用程序和具有关键正常运行时间要求的应用程序。新代码将发布到非活动环境，并在其中进行全面测试。代码经过审核后，团队通常通过调整路由器配置来重定向应用程序流量，使空闲环境处于活动状态。当下一个软件迭代准备就绪时，过程反转。

如果在切换后发现问题，流量可以被引导回仍然运行原始版本的空闲配置。一旦新代码在生产环境中通过测试，团队可以选择更新空闲配置环境中的代码，以提供用于灾难恢复功能的附加度量。17

C 是一种高级通用编程语言，非常适合开发固件或便携式应用程序。C 最初用于编写系统软件，由 Dennis Ritchie 于 20 世纪 70 年代早期在贝尔实验室为 Unix 操作系统开发。C 名列最广泛使用的语言之一，拥有适用于大多数计算机系统的编译器并影响了许多流行语言，特别是 C++。

C# 是一种面向对象的通用编程 (OOP) 语言，用于联网和 Web 开发。C# 被指定为公共语言基础设施 (CLI) 语言。1999 年 1 月，荷兰软件工程师 Anders Hejlsberg 组建了一个团队，开发 C# 作为 Microsoft 的 NET 框架的补充。最初，C# 作为面向对象的类 C 语言 (Cool) 开发。为了避免潜在的商标问题，更改了实际名称。2000 年 1 月，NET 作为 C# 发布。其 NET 框架促进了多种 Web 技术。

C++ 是一种面向对象的通用编程 (OOP) 语言，由 Bjarne Stroustrup 开发，是 C 语言的延伸。因此，可以用“C 风格”或“面向对象的风格”编写 C++ 代码。 在某些情况下，它可以通过两种方式编码，因此是一个有效的混合语言示例。C++ 被认为是中级语言，因为它封装了高级和低级语言功能。最初，该语言被称为“带类的 C 语言”，因为它具有 C 语言的所有属性与“类”的附加概念。 不过，它于 1983 年被重命名为 C++。

网络： 架构（续）

云计算架构是指云计算所需的组件和子组件。这些组件通常包括前端平台（胖客户端、瘦客户端、移动设备）、后端平台（服务器、存储）、基于云的交付和网络（Internet、Intranet、Intercloud）。 简而言之，前端是您看到的部分，后端是幕后发生的计算。

云服务可通过 Internet 公开或私下交付，也可以通过 Intranet 传送到公司的网络中。有时，组织会结合使用两种方式。

无论实际的“云”位于何处（公司自己的数据中心或服务提供商的数据中心），云计算都通过联网来方便地按需访问共享的计算资源池，如网络、存储、服务器、服务和应用程序。通过使用虚拟化，可以根据需要快速轻松地预配置和发布这些资产。

由于虚拟化和虚拟机管理程序等技术，许多虚拟服务器可能存在于单个物理服务器上。这些技术支持其他云订阅，如平台即服务 (PaaS)、基础设施即服务 (IaaS) 等。

前端平台

前端是客户端、客户或用户可见的一端。前端部分包括用户界面，以及用于访问云系统的客户端计算机系统或网络。

后端平台

后端部分位于服务提供商使用的一端。其中包括各种服务器、计算机、数据存储系统、虚拟机和程序，所有这些共同构成计算服务云。后端还负责提供安全机制、流量控制和协议，用于连接网络计算机进行通信。

基于云的交付

如今，云计算服务无处不在。例如，如果您使用 QuickBooks 进行会计工作，使用 Expedia 来安排旅行，或者使用 Netflix 来观看喜爱的电视剧，那么您就是云计算用户。这些都是公司或个人可以购买的订阅示例，这些订阅使他们能够使用该软件（通常称为软件即服务，简称 SaaS）。
